# Política de Privacidade - ZZLink

**Última atualização:** 16 de fevereiro de 2025

O aplicativo ZZLink ("App") é desenvolvido e mantido por MutumSoft. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários.

---

## 1. Sobre o App

O ZZLink é um aplicativo gratuito que permite criar links do WhatsApp (wa.me) a partir de números de telefone. O número digitado é usado exclusivamente no próprio dispositivo para gerar o link — **nenhum número de telefone é enviado, armazenado ou processado em nossos servidores**.

---

## 2. Dados que Coletamos

### 2.1 Dados que NÃO coletamos

- **Números de telefone digitados:** os números são processados localmente no dispositivo e não são enviados para nenhum servidor.
- **Contatos do dispositivo:** o App não acessa sua lista de contatos.
- **Localização:** o App não acessa sua localização.
- **Fotos, arquivos ou mídia:** o App não acessa seus arquivos.
- **Nome, e-mail ou dados pessoais identificáveis:** não solicitamos nem coletamos essas informações.

### 2.2 Dados coletados automaticamente

O App utiliza serviços do Firebase (Google) para melhorar a experiência do usuário e a estabilidade do aplicativo. Os seguintes dados são coletados de forma automática e anônima:

**Firebase Analytics:**

- Eventos de uso do App (abertura, ações realizadas como copiar link, compartilhar, abrir WhatsApp)
- Código de área (DDD) do número digitado (apenas os 2 primeiros dígitos, para fins estatísticos de região)
- Duração aproximada da sessão
- Tipo de dispositivo, versão do sistema operacional e versão do App

**Firebase Crashlytics:**

- Relatórios de falhas e erros do App
- Informações do dispositivo (modelo, versão do sistema operacional)
- Stack traces de erros para diagnóstico

**Firebase Authentication:**

- Um identificador anônimo (UID) é gerado automaticamente para cada instalação do App. Este identificador não está vinculado a nenhuma informação pessoal do usuário e serve apenas para associar dados de analytics e crash reports a uma mesma instalação.

---

## 3. Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para:

- **Melhorar o App:** entender quais funcionalidades são mais utilizadas
- **Corrigir problemas:** identificar e resolver falhas e erros
- **Estatísticas agregadas:** entender a distribuição geográfica de uso por região (DDD)

**Não utilizamos os dados para:**

- Identificar usuarios individualmente
- Enviar publicidade ou comunicações de marketing
- Compartilhar ou vender dados a terceiros
- Criar perfis de usuário

---

## 4. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos dados dos usuários a terceiros, exceto conforme descrito abaixo:

- **Firebase (Google):** os dados de analytics e crash reports são processados pelos serviços do Firebase, operados pelo Google. O Google processa esses dados de acordo com sua própria política de privacidade, disponível em https://policies.google.com/privacy

---

## 5. Armazenamento e Segurança

- Os dados de analytics e crash reports são armazenados nos servidores do Firebase (Google) com criptografia em trânsito (HTTPS/TLS).
- Nenhum dado é armazenado localmente no dispositivo além das preferências básicas do App.
- O identificador anônimo (UID) é gerado pelo Firebase e não contém informações pessoais.

---

## 6. Retenção de Dados

- Os dados de analytics são retidos pelo Firebase por um período padrão de 14 meses e depois são automaticamente excluídos.
- Os dados de crash reports são retidos por 90 dias.
- O identificador anônimo persiste enquanto o App estiver instalado. Ao desinstalar o App, uma nova instalação gerará um novo identificador.

---

## 7. Seus Direitos

Você pode:

- **Desinstalar o App** a qualquer momento, o que interrompe toda coleta de dados.
- **Solicitar a exclusão de dados** entrando em contato conosco pelo e-mail abaixo.
- **Desativar a coleta de analytics** desabilitando o compartilhamento de dados de uso nas configurações do seu dispositivo Android.

---

## 8. Público-Alvo

O ZZLink não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente dados de crianças menores de 13 anos. Se tomarmos conhecimento de que coletamos dados de uma criança menor de 13 anos, tomaremos medidas para excluir essas informações o mais rápido possível.

---

## 9. Permissões do App

O App solicita apenas a seguinte permissão:

- **Internet (android.permission.INTERNET):** necessária para abrir links do WhatsApp no navegador e para o funcionamento dos serviços Firebase (analytics e crash reports).

---

## 10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quaisquer alterações serão publicadas nesta página com a data de atualização revisada. Recomendamos que você revise esta política periodicamente.
